Task killer for screen off

This app will kill the application that has been specified at the time of the screen off.
It is intended to prevent battery drain during sleep.

Specify a kill target app

Please put the check from the list of apps.

However, depending on the apps...
  • That can not be stopped
  • Those that automatically restart
  • Those that affect the system when you stop
Please note that there is such.

Process monitor

It has the ability to monitor the execution state of the process in order to discover the app running unnecessarily.

Please enable the "Activate process monitor" item from the settings screen.

Note: Process monitoring is itself will drain the battery. Please be disabled if there is no need.

If enable the process monitoring, graph of CPU usage is displayed in the apps list. (Graph 10% or more is hidden.)

All processes, including the system is displayed when you showing the "PROCESS" tab.

How to reading CPU graph

Normally it shows the high utilization system_server process.
That's because it is available from other apps in a common feature of the OS.

Resident app that is running at regular intervals may have indirectly battery draining. Even if the CPU utilization is low.

During sleep the CPU is slower. Therefore it looks like CPU utilization has risen, but it is normal.
